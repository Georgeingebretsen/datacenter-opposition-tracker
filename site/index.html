About This Project

Data Sources

Compiled from Data Center Watch, Robert Bryce's reporting, FracTracker Alliance, Change.org petitions, local news coverage, and direct research. AI systems were used to help synthesize and structure this database from public sources. Entries may contain errors.

Contribute

Know of a datacenter fight not listed here? Have corrections or updates? This is an open-source project and contributions are welcome. Download the data using the buttons above and submit additions, or reach out to george.ingebretsen@gmail.com.
